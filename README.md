## **Set up a venv:**

1. create the venv:
```bash
python3 -m venv <env-name>
```

2. activate the env:
```bash
source <env-name>/bin/activate
```

3. show which env you are using atm (optional):
```bash
echo $VIRTUAL_ENV
```

**Install all packages**
```bash
pip3 install -r requirements.txt
```
